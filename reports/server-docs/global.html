(async) connection(socket)

This is the entry point for clients when they reach the Syncopate website. A new socket is established and a unique ID is generated for the user and stored on the database. The user has the ability to create, join, or leave rooms using their socket connection. There is also a test socket listener to test sending "messages" between a group of users in a room.
